# `hour-second-converter`

---

<br>

## Description

Script to switch a string between `HH:MM:SS` and `<number>s` (second).

---

<br>

## How to use

### HH:MM:SS to seconds

```shell
$ node hour-second-converter.js 00:01:40
00:01:40 -> 100s
```

### Seconds to HH:MM:SS

```shell
$ node hour-second-converter.js 100s
100s -> 00:01:40
```
